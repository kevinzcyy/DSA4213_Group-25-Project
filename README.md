# Bridging NER and Sentiment Analysis: Summarising Financial Events

This project explores the connection between **Named Entity Recognition (NER)** and **Sentiment Analysis (SA)** in financial headlines.
By identifying key entities (e.g., companies, people, instruments) and analyzing sentiments around them, the project aims to generate concise summaries of financial events. This provides users with a clearer understanding of market-relevant information extracted from unstructured text.
